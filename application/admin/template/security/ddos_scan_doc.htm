  1. -------- 【宝塔定时扫描教程】 --------
  2. 1、定时扫描接口：点击复制 如扫描出风险，自动以邮箱方式通知。（前往配置邮箱信息）
  3. 2、定时扫描范围：系统文件；（暂不支持扫描附件目录，由于图片、附件等文件可能过多，容易引起超负荷扫描）
  4. 3、以下提供两种定时执行方式（防止服务器压力过载，每天最多执行一次，建议计划任务里设置每天或每几天一次）
  5. 方式一（简易版）：在宝塔面板添加任务类型[访问URL-GET]，可能会超时报404

  6. 方式二（稳定版）：在宝塔面板添加任务类型[Shell脚本]
